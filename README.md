# 🚀 Skill-Based Team Formation & HR Management System

## 📌 Overview

This project is a **Skill-Based Team Formation and HR Management System** built using modern web technologies.
It helps organizations efficiently manage employees and automatically form teams based on required skills.

---

## 🎯 Features

* 👨‍💼 Employee Management (Add, Update, Delete)
* 🧠 Skill-Based Team Formation
* 📊 Performance & Rating Tracking
* 📁 Project Allocation System
* 📅 Availability & Experience Tracking
* 🔍 Smart Search & Filtering
* 📌 Dashboard for HR Insights

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Local Storage / JSON
* Styling: CSS / Tailwind (if used)
* Version Control: Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Techsham1/Skill-based-Team-Formation-System-and-HR-Management-System.git
```

### 2️⃣ Navigate to project folder

```
cd Skill-based-Team-Formation-System-and-HR-Management-System
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the project

```
npm start
```

---

## 💡 How It Works

* Employees are stored with their **skills, experience, and availability**
* HR can create a project with required skills
* System automatically suggests best-fit team members
* Performance and ratings help in better future allocation
